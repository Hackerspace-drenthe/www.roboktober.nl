# Release Notes - 2026-07-08

Deze release bundelt de nieuwste wijzigingen rond account-flow, editor-UX en governance.

## Highlights

### 1. Teamregistratie nu account-verplicht
- Nieuwe teamaanmelding vereist een ingelogd account.
- Nieuwe teams worden direct gekoppeld aan de ingelogde gebruiker (`captain_user_id`).
- Registratieflow sluit nu beter aan op ownership en permissies.

### 2. Mijn aanmelding opent direct eigen bewerkomgeving
- Route `/app/aanmelding/wijzigen` is nu login-protected.
- Ingelogde users worden automatisch doorgestuurd naar hun eigen editpagina.
- Bij ontbrekende koppeling is er een duidelijke fallback met retry/CTA.

### 3. Editor UX verbeterd (content + media)
- Herbruikbare formatting-toolbar toegevoegd (o.a. bold, italic, headings, lijsten, link, quote, code).
- Resource-insert flow verbeterd voor media in content (video/STL/afbeeldingen).
- Teamfoto UX verbeterd met live preview in aanmeldflow.
- In bewerkflow is er nu een duidelijke huidige-vs-nieuwe fotovergelijking.

### 4. Clarify-first AI governance aangescherpt
- Bij onduidelijke taken: eerst gerichte verduidelijkingsvragen.
- Bij middel/hoge complexiteit: minimaal 1 bevestigingsvraag voor codewijzigingen.
- Uitzondering blijft mogelijk bij expliciete opdracht om direct uit te voeren.

## Technische samenvatting
- API-auth en registratie-flow verder geharmoniseerd.
- Frontend route-guards en redirects aangescherpt.
- Herbruikbare editor-composables/componenten uitgebreid.
- Procesinstructies voor AI-assistentie vastgelegd op project- en userniveau.
